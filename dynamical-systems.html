[02.2] Technical Architecture

Qt 6 Application Shell

Built with Qt 6 Widgets, the application uses QMainWindow for layout, QOpenGLWidget for custom rendering, and QWidget-based panels for parameter controls and overlays.

C++ Backend Simulation

The core solver is a C++ implementation of RK4 integration with configurable time step and substep handling for accurate trajectory evolution in nonlinear differential equations.

Visualization & Data Flow

Simulation state flows into the Qt rendering layer via scene buffers, while overlay modules compute energy, Lyapunov divergence, and Poincaré data on the fly for real-time visualization updates.

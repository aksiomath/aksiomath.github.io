Maze & Pathfinding Visualizer

A Qt desktop app that combines advanced maze generation and graph search visualization with interactive terrain costs, algorithm selection, and heatmap insights.

[02.1] Key Features

Maze generation

Maze Generation Algorithms

Supports DFS Backtracking, Prim's, Kruskal's, and Wilson's algorithms with animated carving and seeded reproducibility.

Path search

Pathfinding Algorithms

Includes BFS, Dijkstra, and A* with real-time visualization of search progress, frontier expansion, and final path reconstruction.

c = 1,3,5

Interactive Terrain Costs

Right-click cells to assign terrain difficulty values of 1, 3, or 5, influencing weighted pathfinding and exploration behavior.

Heatmap

Visualization Controls

Animated generation and search with adjustable speed, heatmap overlays, start/goal highlights, and status updates.

+
[02.2] Technical Architecture

Qt 6 UI Framework

Built with Qt 6 Widgets and QGraphicsScene for flexible grid rendering, interactive cell controls, and a sidebar for algorithm selection.

C++ Algorithm Engine

Implements graph-based maze generation and pathfinding in C++ with efficient neighbor tracking, parent pointers, and weighted cost handling.

Real-Time Animation

Dual animation engines manage maze carving and path search independently, updating the Qt render loop while preserving interactive controls.

+
[02.3] Product Summary

Visual algorithm learning

An interactive visualization tool for teaching and exploring maze generation and pathfinding with clear animation, cost-aware routing, and algorithm comparison.

highlights

  • Four maze generation methods with animated construction
  • Three pathfinding strategies with clear frontier visualization
  • Terrain cost editing and weighted search behavior
  • Heatmap overlay and progress status feedback

interaction

Users can place start and goal points, toggle overlays, and step through generation and search at their own pace.

[03.0] Interesting facts

Some maze algorithms build paths by exploring from a single starting point.

// DFS maze generation carves a perfect maze with no loops and a unique path.
// Prim and Kruskal create mazes by joining walls while avoiding cycles.
// A* uses a heuristic to guide the search toward the goal efficiently.
// Weighted terrain makes pathfinding resemble real-world navigation.
